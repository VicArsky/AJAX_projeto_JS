Async

O Async pode ser colocado antes de uma função. Esta função retorna uma promise.
Os valores são agrupados na promise automaticamente.

Exemplo 1

async function f() { return Promise.resolve(1); } f().then(alert);

Retorna:
1

Await

Funciona apenas em funções async. Este comando espera até a promessa terminar e
retorna o resultado. Await suspende a função enquanto a promise termina, e
depois continua com o resultado da promise.

Exemplo 2

async function f() {
let promise = new Promise(resolve/reject) => {
setTimeout(() => resolve("Finalizado!"), 3000)
});
let result = await promise;
alert(result);
}
f()

Retorna:
Finalizado!

Vantagens

  • A sintaxe é direta e fácil de entender.

  • É fácil corrigir os erros em código assíncrono, pois o código vira síncrono
    durante a correção.


Desvantagens

  • Precisa tomar cuidado com os erros, se passarem despercebidos podem
    ser mais dificeis de corrigir.

  • O await pode bloquear a execução do código, se for usado em loops.
