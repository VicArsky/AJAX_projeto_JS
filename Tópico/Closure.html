Closure

Um closure permite uma função acessar variáveis do próprio escopo externo até
quando o escopo tiver fechado.

Exemplo:

function outraFunction() {
let outravariable = "Elemento de outra função."
function interiorFunction() {
console.log(outravariable);
}
return interiorFunction;
}

let closureExemplo = outraFunction();

closureExemplo;

Vantagens

  • Closures permite a criação de variáveis privadas e métodos,
    aumentando a segurança dos dados e prevenindo acesso não
    permitido ou modificação.

  • Oferece flexibilidade no design do código ao deixar a criação
    de função especializadas e comandos feitos para requisitos
    específicos.

  • Encapsula padrões comuns e comandos em funções
    reusáveis.


Desvantagens

  • Pode aumentar o consumo de memória.

  • O uso impróprio de closures pode levar a vazamentos de
    memória se referências para variáveis de escopo externo
    forem retidas inadivertidamente.

  • Torna complexo corrigir erros, em cenários onde
    closures estão agrupadas.
