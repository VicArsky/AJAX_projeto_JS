Ajax

É uma forma de usar JavaScript para pedir informações de um servidor sem precisar
atualizar a página inteira. Isso deixa o site mais rápido e faz com que só
algumas partes mudem, em vez de carregar tudo de novo.

Exemplo:

function buscarTexto() {
var xhr = new XMLHttpRequest();
xhr.open('GET', 'exemplo.txt', true);
xhr.onload = function() {
if (xhr.status === 200) {
document.getElementById('conteudo').innerText = xhr.responseText;
}
};
xhr.send();
}

Vantagens do Ajax:

  • Atualização sem recarregar a página: Permite que partes de uma página sejam atualizadas sem precisar recarregar tudo, o que deixa o site mais rápido e fluido.

  • Melhora a experiência do usuário: Os usuários podem interagir com a página enquanto os dados são carregados em segundo plano, sem ficar esperando.

  • Menos dados transferidos: Só os dados necessários são enviados e recebidos, reduzindo o uso de banda e tempo de carregamento.

Desvantagens do AJAX:

  • Compatibilidade: Nem todos os navegadores antigos suportam AJAX corretamente, o que pode causar problemas em sites que dependem dele.

  • Pode ser difícil de depurar: Como as requisições são assíncronas, pode ser mais complicado encontrar e corrigir erros.

  • SEO limitado: O conteúdo carregado dinamicamente via AJAX pode ser mais difícil para motores de busca, como Google, entenderem, o que pode prejudicar a visibilidade do site em buscas.

  • Requer conhecimento adicional: Para usar AJAX de forma eficaz, é preciso saber lidar com JavaScript, JSON, e lidar com APIs.
