Hoisting

Hoisting é um comportamento do JavaScript que faz com que algumas declarações (como variáveis e funções)
sejam "jogadas para cima" do código antes dele ser executado. Isso significa que você pode usar uma variável
ou função antes mesmo de declará-la no código. (é como se o JavaScript lesse as declarações do seu código
antes de começar a rodar e se “lembra” onde você declarou variáveis e funções e coloca essas
declarações lá no topo, mas não o valor delas.)

Exemplo:

console.log(fruta);
var fruta = "banana";
console.log(fruta);

var fruta;
console.log(fruta);
fruta = "banana";
console.log(fruta);
Retorna:
undefined

banana
