Arrow Functions

Arrow Functions (funções de seta) são uma maneira mais curta e moderna de escrever funções em JavaScript,
usando uma "seta" (=>). Elas foram introduzidas para deixar o código mais limpo e fácil de ler.

Exemplo:

Função normal

function Carro() {
this.velocidade = 0;
setTimeout(function() {
this.velocidade++;
console.log(this.velocidade);
}, 1000);
}
const meuCarro = new Carro();
Com Arrow Function

function Carro() {
this.velocidade = 0;
setTimeout(() => {
this.velocidade++;
console.log(this.velocidade);
}, 1000);
}
const meuCarro = new Carro();

Diferenças principais em relação às funções normais:

Sintaxe mais curta:
  • As Arrow Functions são mais compactas, deixando o código mais simples.


  • Em vez de escrever:
    function soma(a, b) {
    return a + b;
    }

    Com arrow function fica assim:

    const soma = (a, b) => a + b;


  • Se a função tiver apenas uma linha de código (um retorno), você nem precisa
    da palavra return ou das chaves {}.


this diferente:
  • Em funções normais, o valor de this pode mudar dependendo de como a
    função é chamada (por exemplo, ele pode se referir ao objeto que
    chamou a função).


  • Já nas Arrow Functions, o this é "fixo". Ele pega o valor de this do local onde
    a função foi criada e não muda, mesmo se for chamada de outra forma.

Exemplo:

  • Com uma função normal:
function Carro() {
this.velocidade = 0;
setTimeout(function() {
this.velocidade++;
console.log(this.velocidade);
}, 1000);
}

const meuCarro = new Carro();
  • Com arrow function:
function Carro() {
this.velocidade = 0;
setTimeout(() => {
this.velocidade++;
console.log(this.velocidade);
}, 1000);
}

const meuCarro = new Carro();
  • Nesse caso, a Arrow Function é útil porque ela mantém o this dentro do carro, então a velocidade aumenta corretamente.

Não podem ser usada como funções construtoras:

  • Funções normais podem ser usadas para criar objetos usando new:
function Pessoa(nome) {
this.nome = nome;
}

const eu = new Pessoa("Yuri");
console.log(eu.nome);
const Pessoa = (nome) => {
this.nome = nome;
};

const eu = new Pessoa("Yuri");
