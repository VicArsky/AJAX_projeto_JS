Fetch API

O fetch api pega um recurso de um servidor e devolve uma promise.

Exemplo:

getTexto("página.txt");

async function getTexto(arquivo) {
let x = await fetch(arquivo);
let y = await x.texto();
minhaTela(y);
documentgetElementById("teste").innerHTML = y;
}

Vantagens

  • Fetch é nativo do JavaScrip API, não precisa de bibliotecas adicionais.

  • Usa promises, facilita pedidos assíncronos.

  • A configuração permite usar em casos simples.


Desvantagens

  • Não dá para usar em navegadores antigos.

  • O uso é complicado para configurações avançadas, como timeout.

  • As respostas devem ser manualmente transformadas para o JSON.
